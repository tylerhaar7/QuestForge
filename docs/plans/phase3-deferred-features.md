# Deferred Features — To Build If Worth the API Costs

Features from the full spec that were intentionally deferred from the Phase 3 playable loop. Build these incrementally if the core game loop proves fun and cost-effective.

---

## Screens

### Camp Screen (`app/game/camp.tsx`)
- Triggers after 3-4 encounters, boss fights, or critical HP
- Activities: talk to companion, rest (full HP/spell recovery), cook (temp buffs), train (XP), explore camp (random events), give gift (approval boost), journal, companion banter
- Warm amber gradient, campfire ambient sound
- Companion dialogue gated by approval score

### Threshold Screen (`app/game/threshold.tsx`) — Death Hub
- Hades-inspired death mechanic: death is not game over
- The Keeper (mysterious NPC) greets player, references how they died
- Three options: return to world, explore threshold (hidden lore), ask for guidance
- Companions react per personality (Korrin determined, Sera humor, Thaelen contemplative)
- Death count tracked, unlocks meta-progression

### Journal Screen (`app/game/journal.tsx`)
- Auto-generated journal from game events
- Tabs: All, NPCs, Quests, Locations, Lore, Decisions
- Searchable, parchment-styled cards, pin important entries
- Player can add personal notes

### Codex Screen (`app/game/codex.tsx`) — Living Lore Encyclopedia
- Certain words in narrative text highlighted with gold underline
- Tap opens bottom sheet with lore entry
- Categories: People, Places, Factions, Creatures, Items, History
- Auto-generated by Claude when new lore introduced

### Adventure Map (`src/components/game/AdventureMap.tsx`)
- Slay the Spire-style branching node map
- Node types: combat, elite, boss, rest, merchant, mystery, social, treasure
- Vertical scrolling, gold paths for available routes, tap to select
- Replaces linear narrative flow

## Edge Functions

### Session Recap (`supabase/functions/session-recap`)
- "Previously on..." recap when loading a campaign
- Fetches last 20 turns, calls Haiku for dramatic 3-4 sentence summary
- Shows key points, current threat, companion status
- Cost: ~$0.001 per recap

### Image Generation (`supabase/functions/generate-image`)
- Character portraits at creation
- Scene art at major story beats
- NPC portraits on first meeting (premium only)
- Free tier limits: 1 portrait + 3 scene images per session
- Cost: ~$0.50-1.00 per active campaign per month

## AI Prompt Enhancements

### Plot Thread Management
- Active threads tracked in Supabase plot_threads table
- Every 15-20 turns, at least one thread referenced or advanced
- After 30+ turns without advancement, force an event
- Never more than 5 active threads
- Thread updates returned in AIResponse

### Dynamic Difficulty
- Track: win_rate, avg_hp_at_combat_end, deaths, retry_rate
- Auto-adjust: >80% win rate → harder, <40% → easier
- Three player preferences: story, balanced, hardcore
- Never tell the player it's adjusting

### Threshold Mode Prompt
- Specialized prompt for death hub conversations
- The Keeper references specific death cause
- Reveals lore only available through death
- Philosophical, wry tone — death as teacher

## Companion System (Full)

### Relationship Stage Transitions
- 7 stages: hostile (0-10) → cold (11-25) → neutral (26-45) → warm (46-60) → friendly (61-75) → close (76-90) → devoted (91-100)
- Milestone events:
  - Below 20: Companion confronts player, threatens to leave
  - Above 65: Shares personal backstory
  - Above 80: Personal quest unlocks
  - Above 95: Grants unique ability

### Personality-Driven Approval
- Thaelen: +nature, +patience, +wisdom. -destruction, -haste, -fire
- Sera: +cleverness, +loyalty, +street smarts. -naivety, -authority, -cruelty
- Korrin: +honor, +protection, +honesty. -deception, -cowardice, -harming innocents

### Companion Banter
- Two companions converse without player involvement
- Based on relationship stages and recent events
